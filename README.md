
#### Instalar express
```
npm install express
```

#### Instalar nodemon
```
npm install --save-dev nodemon
```

#### Instalar mongoose
```
npm install mongoose --save
```

#### Instalar sucrase
```
npm install --save-dev sucrase
```

#### Instalar bcrypt
```
npm i bcrypt
```

#### Instalar jsonwebtoken
```
npm install jsonwebtoken
```
